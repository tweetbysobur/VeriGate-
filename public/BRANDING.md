# VeriGate Brand Assets

## Logo Files

### Primary Mark
- **logo-mark.svg** — VG monogram in gradient badge (256×256px, scalable)
  - The checkmark IS the V, bridging into the G ring
  - Deep Indigo #5B5BF7 → Electric Purple #7C4DFF
  - Use for app icon, favicon, badges

### Wordmark
- **logo-wordmark.svg** — "VeriGate" with the mark (320×80px, scalable)
  - Horizontal lockup for headers, hero, branded materials
  - "Gate" in gradient; "Veri" in dark text
  - Primary use: headers, landing page hero

### Glyph (No Badge)
- **logo-glyph.svg** — VG monogram alone, no badge (256×256px, scalable)
  - The check-V bridging into the G, gradient stroke only
  - Use for light/dark surface overlays, embeds

## Brand Colors

| Name | Hex | Usage |
|------|-----|-------|
| Deep Indigo | #5B5BF7 | Primary gradient start |
| Electric Purple | #7C4DFF | Primary gradient end |
| White | #FFFFFF | On dark backgrounds |
| Dark Ink | #0B1020 | On light backgrounds |

## Typography

- **Wordmark:** System sans-serif (Geist, -apple-system)
- **Weight:** 600 (semibold) for "Veri", gradient for "Gate"

## Tagline

**Verified. Compliant. Trusted.**

## Usage Guidelines

1. **Always maintain the check-to-G bridge** — it's the core design concept
2. **Minimum size:** 32×32px (favicon), 64×64px (icon)
3. **Clear space:** At least 4px padding around the mark
4. **Color:** Use gradient preferred; monochrome (black or white) acceptable for constrained spaces
5. **On dark backgrounds:** Logo mark with white monogram
6. **On light backgrounds:** Logo mark with inverted fill, or gradient glyph

## File Locations

All SVG files are in `/public/` and can be used directly or converted to PNG/PDF.
